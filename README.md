# HogwartsAR

Demo: https://www.youtube.com/watch?v=2Ct0H9CzSyA

### Prerequisites
- iOS 11 / ARKit compatible 
- Xcode 9 beta

### To get started
1. clone project
2. extract `Assets/Hogwarts.obj.zip` to `Assets/art.scnassets`
3. open `HogwartsAR.xcodeproj`
4. Build to iOS 11 device
